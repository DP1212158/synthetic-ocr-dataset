# 01_报纸页

本目录属于 `v12` JSON 结构树驱动分类版，类别为 `报纸页`（`newspaper_page`），共 6 张。

## 目录

- `html/`：HTML 模板
- `images/`：渲染图片
- `labels/`：标签 JSON
- `metadata/`：该类别 manifest
- `reports/contact_sheet.jpg`：该类别联系图
- `reports/contact_sheet_with_boxes.jpg`：带标签框联系图

## 模板

- 结构树来源：`newspaper_layout_templates_v1.json`
