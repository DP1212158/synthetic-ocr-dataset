# 藏语版面模板调研库

本目录用于保存藏语造数据工程的版面模板参考资料。

目标：

1. 为 12 类版面分别建立可追溯的参考来源。
2. 记录每类版面的关键结构元素。
3. 作为后续合成脚本更新和人工验收的依据。

## 当前模板规格

- `layout_prompts_v2.json`
  - 结构化 prompt 主入口。
  - 覆盖当前藏语工程 12 类，每类 6 个模板族，共 72 个模板规格。
  - 后续如果替换或扩写 prompts，优先更新这个 JSON。

- `newspaper_layout_templates_v1.json`
  - 报纸页专用结构树模板。
  - 覆盖报纸页 6 套模板族。
  - 该文件已经用于 `v12`。

- `reading_pages_layout_templates_v1.json`
  - 阅读型页面结构树模板。
  - 覆盖 `书籍页 / 教科书页 / 杂志期刊页`，每类 6 套模板族。
  - 该文件已经用于 `v12`。

- `12类版面多样性Prompt设计稿.md`
  - 人类可读的 prompt 设计稿。
  - 用于讨论和人工审阅，不作为脚本强校验入口。

- `12类版面模板来源总表.md`
  - 版面参考来源和适用说明。

## 结构化 prompt 校验

校验脚本：

- `../03_合成数据生成_20260630/scripts/validate_layout_prompts.py`

示例命令：

```bash
python3 ../03_合成数据生成_20260630/scripts/validate_layout_prompts.py \
  layout_prompts_v2.json \
  --category-plan ../03_合成数据生成_20260630/v11/metadata/category_plan.json
```

校验目标：

- 12 类顺序和当前生成版本一致。
- 每类必须有 6 个模板。
- 共 72 个模板。
- 每个模板必须包含 prompt、版面元素、随机化参数和 QA 注意。
- 禁止混入已下线旧类别。
