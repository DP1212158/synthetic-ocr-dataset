## 对项目的意义

- 适合：公开站点正文、短段落、横排文本填充。
- 不足：题材可能偏百科或偏文献；很多页面较短；不同语言站点需要单独评估脚本占比。
- 建议：先用 API 小批量验证模板流程；后续若需要更大文本池，再扩展到其他公开文本源。
