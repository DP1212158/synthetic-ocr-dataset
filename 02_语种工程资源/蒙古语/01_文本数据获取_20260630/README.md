# 文本数据获取记录

本目录用于记录少数民族语言文本数据的采集脚本、配置、样本数据和评估报告。

## 目录结构

- `scripts/`：可复用采集脚本和一键运行脚本。
- `configs/`：采集参数、代理配置和数据源说明。
- `data/`：按语言和来源归档的采集结果。
- `reports/`：人工整理后的阶段性判断报告。
- `logs/`：运行日志。

## 当前阶段

先验证蒙古语 Wikipedia 是否足够作为后续“版面模板造数据”的文本来源。重点不是一次性爬满，而是确认：

- 网络和接口是否稳定；
- 能否保存结构化文本；
- 文本字符类型是否符合项目目标；
- 数据量是否足够支撑批量生成。

## 快速运行

```bash
bash /Users/dp/Downloads/中央民族大学合作项目/文本数据获取_20260630/scripts/run_mn_wikipedia_sample.sh
```

输出会保存到：

```text
/Users/dp/Downloads/中央民族大学合作项目/文本数据获取_20260630/data/mn_wikipedia_api_sample_20260630
```

## 注意

Wikipedia 文本需要保留来源和许可证说明。后续用于合成版面数据时，建议继续保留 `source_url`、`title`、`license_note` 等字段。
