# 合成OCR数据集_交付版

本目录是从 `/Users/dp/Downloads/中央民族大学合作项目` 中独立整理出的合成 OCR 数据交付工程。目录采用复制方式生成，不依赖原始工程目录中的历史版本。

## 核心内容

- `01_最终VL2数据/`：7 个语种的最终 VL2 数据，每个语种 12 类 x 6 张，共 72 张。
- `02_语种工程资源/`：每个语种的文本源、模板调研、生成配置和脚本副本。
- `03_通用生成pipeline/`：多语种生成、渲染、增强和 QA 的通用脚本。
- `04_共享图片资产/`：用于替换图片占位符的 `random120-coco2014` 图片池。
- `05_总报告与索引/`：VL2 交付索引和 QA 汇总。

## 当前交付版本

当前统一交付版本为 `VL2`。每个语种目录下均包含：

- `html/`
- `images/`
- `labels/`
- `metadata/`
- `reports/`

每个版本根目录包含总览图、带框总览图、QA 报告、语言混合报告、语言有效性报告、阅读顺序报告、图片资产报告和增强质量报告。

## 快速入口

- QA 汇总：`05_总报告与索引/VL2_QA_汇总.md`
- JSON 索引：`05_总报告与索引/VL2_交付索引.json`
- 通用脚本：`03_通用生成pipeline/多语种合成数据生成_20260702/scripts/`

## 注意

本数据是合成补充数据，不替代真实采集数据。自动 QA 可以验证脚本残留、标签越界、密度、阅读顺序、图片占位、增强记录等工程指标，但不能完全替代母语专家对语义自然度和语法正确性的最终审核。
